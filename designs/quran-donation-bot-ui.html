Premium Telegram Charity UX

Quran Donation Bot

A complete Telegram-first donation journey for sponsoring Qurans for Muslim children in rural Tigray. The design is intentionally clean, respectful, and production-ready, with strong visual hierarchy, clear payment guidance, and visible trust signals.

450 Birr The price per Quran is repeated at every key moment so the donor always understands the amount.
Simple Flow Start, choose amount, select payment, upload receipt, wait for review, and receive approval.
Trust Cues Account cards, pending badges, approval notices, and support access reduce hesitation.

Design principles

Islamic-friendly tone Soft green palette, calm white surfaces, and respectful charity language.
Telegram-native layout Reply keyboards for navigation and inline buttons for focused choices.
Minimal but premium Elegant cards, spacious message bubbles, and visible status hierarchy.
First-time user clarity Every screen gives one obvious next action with no ambiguous branching.

End-to-End Flow

Conversation Journey

1. StartUser sees the mission, price per Quran, update channel, and main menu keyboard.
2. Enter amountThe bot asks for Quran quantity and validates that the message is a positive number.
3. Choose methodThe total is calculated automatically and payment methods appear as inline buttons.
4. PayUser gets a clear account card with provider, account number, account name, and amount due.
5. Submit receiptThe bot waits for an image and blocks unsupported content while in payment-proof mode.
6. Pending reviewThe donor receives a clear pending state so there is no confusion after upload.
7. ApprovedAdmin confirmation closes the loop with gratitude and a fast path to donate again.
